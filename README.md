# ぶーぶーランド 🚒

2さい向けの「はたらくのりもの」知育PWA。タップひとつで のりものの名前・音・お話を楽しめる、**完全オフライン**で動くアプリです。

**公開URL: https://dazoyee.github.io/buubuu/**

## 概要

「ぶーぶーランド」は、保護者と一緒に、またはタップだけでも遊べる2歳児向けの知育アプリです。
しょうぼうしゃ・ぱとかー・ショベルカーなどの はたらくのりものを題材に、音声ファーストで構成しています。
実行時に外部API・バックエンドを一切呼ばず、ネット未接続でも全機能が動作します。

## 特長

- 🚒 **3つのモード**: ①これなあに？（名前あて） ②音・まね遊び（効果音＋オノマトペ） ③みじかいお話（名前差し込み式）
- 🏠 **ガレージ（ずかん）**: 出会ったのりものをコレクション（シルエット→着色）
- 🔊 **音声ファースト**: Web Speech API（日本語・やや遅め）で読み上げ
- 🎨 **アニメ付きSVGイラスト**: のりもの54種を手描きSVGで表現（タイヤ回転・ライト点滅 など）
- 📴 **完全オフライン**: Service Worker で全アセットをキャッシュ
- 👶 **誤操作しにくいUI**: 大きなタップ領域、保護者設定は3秒長押しゲート
- 🧒 **パーソナライズ**: お子さんの名前を端末内に保存し、声かけ・お話に差し込み

## デモ

公開版: **https://dazoyee.github.io/buubuu/**

iPhone の Safari で開き、ホーム画面に追加すると全画面PWAとして使えます（下記手順）。

## 構成ファイル

| ファイル | 役割 |
|---|---|
| `index.html` | アプリ本体（HTML/CSS/JS・全コンテンツ・SVG・音声合成を内包） |
| `manifest.webmanifest` | PWA定義（アイコンはSVG data URI） |
| `service-worker.js` | 全アセットのキャッシュ（オフライン動作） |

## ローカルで動かす

ビルド不要です。Service Worker は `https` か `localhost` でのみ動くため、ファイルを直接開くのではなく簡易サーバーで配信します。

```bash
# 例: Python の簡易サーバー
python3 -m http.server 8000
# → http://localhost:8000 を開く
```

## iPhone でホーム画面に追加

1. iPhone の **Safari** で公開URL（または配信先）を開く
2. 画面下の**共有ボタン** → 「**ホーム画面に追加**」
3. 追加されたアイコンから起動すると全画面PWAで開く
4. 一度開けば **オフラインでも全機能が動作** します

> 起動後の「タッチして はじめる」で音声が有効になります（iOS の自動再生制限に対応）。

## 技術スタック

- 素の HTML / CSS / JavaScript（フレームワーク・ビルドツールなし）
- Web Speech API（音声読み上げ）
- Web Audio API（効果音の合成）
- SVG + CSS アニメーション（のりものイラスト）
- localStorage（名前・ガレージ・既読の保存）
- PWA（manifest + Service Worker）
